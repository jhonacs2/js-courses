## Temas que cubrirás

- `grid-column: start / end` y `grid-row: start / end`
- `span` para abarcar múltiples celdas
- `grid-template-areas` — layout declarativo con nombres
- `grid-area` en cada ítem
- Alineación: `justify-items`, `align-items`, `place-items`
- `justify-self` y `align-self` — alineación individual

## Reto

Crear un layout de revista/blog con `grid-template-areas`: header, sidebar, contenido principal y footer. El artículo destacado debe abarcar 2 columnas.
