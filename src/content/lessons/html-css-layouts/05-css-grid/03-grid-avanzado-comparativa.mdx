## Temas que cubrirás

- Grid implícito vs explícito: `grid-auto-rows`, `grid-auto-columns`
- `grid-auto-flow`: row, column, dense
- Subgrid (introducción) y grid anidado
- Comparativa: Floats (legado) vs Flexbox (1D) vs Grid (2D)
- Combinar Flexbox y Grid en el mismo proyecto
- Práctica: Dashboard Layout (Grid para estructura, Flexbox para componentes)

## Reto

Construir un layout de dashboard con sidebar, header, área de contenido con cards estadísticas, y tabla de datos. Usar Grid para el layout general y Flexbox para los componentes.
