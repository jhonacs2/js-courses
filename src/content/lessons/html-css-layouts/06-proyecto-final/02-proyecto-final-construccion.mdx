## Temas que cubrirás

- Definición del proyecto: portfolio personal, landing page de producto, o blog
- Requisitos obligatorios:
  - HTML semántico completo (header, nav, main, footer)
  - Mínimo 3 secciones diferenciadas
  - Formulario de contacto con validación nativa
  - Layout con Flexbox Y CSS Grid (ambos)
  - Responsive: funcionar en móvil, tablet y desktop
  - Tipografía web (Google Fonts)
  - Transiciones CSS en interacciones
- Sesión de trabajo guiada y revisión de código

## Reto

Avanzar la construcción del proyecto final con estructura HTML completa, layout base con Grid y componentes con Flexbox.
