## Temas que cubrirás

- Selectores combinados: descendiente, hijo directo (`>`), hermano adyacente (`+`), hermano general (`~`)
- Selectores de atributo: `[type="text"]`, `[href^="https"]`
- Pseudoclases: `:hover`, `:focus`, `:active`, `:first-child`, `:nth-child(n)`, `:not()`
- Pseudoelementos: `::before`, `::after`, `::first-line`, `::first-letter`
- Reglas de especificidad: inline > id > clase > etiqueta
- Herencia y la keyword `inherit`

## Reto

Crear una lista de tarjetas de producto donde al hacer `:hover` cambie el fondo, borde y sombra. Usar `::before` para añadir un ícono decorativo a cada tarjeta.
