## Temas que cubrirás

- El modelo de caja: content → padding → border → margin
- Visualizarlo con DevTools de Chrome
- `box-sizing: content-box` vs `border-box`
- Reset universal con `border-box`
- `display: block`, `inline`, `inline-block`
- Colapso de márgenes
- Dimensiones: `width`, `height`, `max-width`, `min-height`
- `overflow`: visible, hidden, scroll, auto
- `border`, `border-radius`, `box-shadow`

## Reto

Maquetar 3 tarjetas de presentación (cards) usando box model, con bordes redondeados, sombra al hover, y un botón inline-block centrado.
