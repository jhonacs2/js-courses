## Temas que cubrirás

- Formas de aplicar CSS: inline, interno (`<style>`) y externo (`.css`)
- Selectores básicos: etiqueta, clase (`.`), id (`#`), universal (`*`)
- Propiedad `color` y `background-color`
- Formatos de color: nombres, hex, rgb(), rgba(), hsl()
- Gradientes básicos: `linear-gradient()`
- Tipografía: `font-family`, `font-size`, `font-weight`, `font-style`
- `text-align`, `text-decoration`, `text-transform`
