## Temas que cubrirás

- Hipervínculos: etiqueta `<a>`, `href`, `target`, `title`
- Rutas relativas vs absolutas, links internos con `#id`
- Imágenes: `<img>`, `src`, `alt`, `width`, `height`
- `<figure>` y `<figcaption>` para imágenes con contexto
- Formatos web: jpg, png, svg, webp — cuándo usar cada uno
- Práctica integradora: mini-web de 3 páginas enlazadas

## Reto

Crear un mini-sitio de 3 páginas (Inicio, Proyectos, Contacto) con navegación entre ellas, imágenes, y estilos CSS unificados desde un solo archivo externo.
