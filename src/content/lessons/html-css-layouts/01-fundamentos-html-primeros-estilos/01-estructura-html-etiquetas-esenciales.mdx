## Temas que cubrirás

- Estructura base de una página HTML
- Anatomía de una etiqueta: apertura, cierre, atributos, contenido
- Etiquetas de texto: encabezados `<h1>`–`<h6>`, párrafos `<p>`, `<strong>`, `<em>`, `<span>`
- Texto especial: `<pre>`, `<code>`, saltos `<br>` y líneas `<hr>`
- Listas ordenadas `<ol>` y desordenadas `<ul>`, listas anidadas
- Tablas básicas: `<table>`, `<tr>`, `<td>`, `<th>`, `<thead>`, `<tbody>`
- Atributos fundamentales: `id`, `class`, `title`, `lang`

## Reto

Crear una página "Sobre Mí" con estructura semántica básica, dos tipos de listas y una tabla con datos personales (hobbies, habilidades, etc.).
