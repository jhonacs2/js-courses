## Temas que cubrirás

- `position: static` (flujo normal), `relative`, `absolute`, `fixed`, `sticky`
- `z-index`: contexto de apilamiento
- `opacity` y `visibility: hidden` vs `display: none`
- Transiciones CSS: `transition`, timing functions
- `transform` básico: `scale()`, `translateX()`, `rotate()`
- Práctica integradora: landing page completa

## Reto

Crear una página con: header fijo (`position: fixed`), sección hero con overlay (`absolute`), formulario de contacto estilizado, y transiciones suaves.
