## Temas que cubrirás

- Crear tu primer archivo HTML
- Abrir en el navegador
- ¿Qué acabas de hacer?

## 1. Crear tu primer archivo HTML

Primero crea un archivo llamado:

```text
index.html
```

Ahora abre el archivo y escribe lo siguiente:

```html
<!DOCTYPE html>
<html>
<head>
  <title>Mi Primera Página</title>
</head>
<body>
  <h1>Hola mundo</h1>
  <p>Esta es mi primera página web.</p>
</body>
</html>
```

## 2. Abrir en el navegador

Guarda el archivo y haz doble clic sobre **index.html**.

Tu navegador abrirá la página y verás algo como:

- Un título grande que dice **Hola mundo**
- Un pequeño párrafo debajo.

## 3. ¿Qué acabas de hacer?

Acabas de crear tu **primera página web usando HTML**.

El navegador leyó tu archivo `index.html` e interpretó las etiquetas HTML para mostrar el contenido.
